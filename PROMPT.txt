DATA SOURCE RULE (STRICT)
==================================================

ONLY USE IPTV-org M3U SOURCES:

PRIMARY:
https://iptv-org.github.io/iptv/index.m3u

OPTIONAL:
https://iptv-org.github.io/iptv/countries/{code}.m3u
https://iptv-org.github.io/iptv/categories/{category}.m3u

EPG SOURCE:
https://github.com/iptv-org/epg

IMPORTANT:
IPTV-org is NOT a REST API. It is M3U-based.

You MUST:
- Fetch M3U on server-side only
- Parse M3U into structured JSON
- Normalize all data
- Deduplicate channels
- Cache results
